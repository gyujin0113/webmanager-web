## 2. 제품 정의 (YC 오피스아워로 도출) → 상세: artifacts/01
### 문제
코딩 모르는 소상공인에게 맞춤 홈페이지를 만들어준 뒤, 배포 후 자잘한 수정 요청("영업시간
바꿔줘" 등)이 끝없이 들어와 사람이 손으로 쳐내느라 시간이 녹는다. 고객은 개발자에게 카톡하고
며칠 기다린다. **나 자신이 이미 이 고통을 겪는 첫 사용자** (미용실 원장님 + 진엽 형 회사 사이트를
실제로 만들어 납품했고, 그들이 내 폰으로 수정 요청을 보냄).

### 진짜 제품 (재정의)
> **맞춤 홈페이지 제작 + 월 구독형 "무제한 수정" 유지보수.**
> AI는 기능이 아니라, 무제한 수정을 수익성 있게 만드는 **마진 엔진**.
> (무제한 수정을 사람이 손으로 하면 망함. AI가 치면 한계비용≈0 → 구독 무제한 판매 가능.)

### 고객 / 시장
- 돈 내는 고객 = 코딩 모르는 끝단 사장님. 이중 수혜(사장님=빠른 수정+안심 / 3&D=노동해방+구독매출).
- 진짜 경쟁자 = "개발자한테 카톡하고 기다림" + 노코드 산업 전체(Wix·아임웹·카페24, 다 AI 붙는 중).
- 차별점 = **묶음**: "우리가 *맞춤으로* 만들어주고(Wix는 셀프) + AI 셀프수정을 얹는다."

### MVP 접근법 (채택: A+C 하이브리드)
- **A. 사람이 게이트, AI가 손발**: 요청→AI가 변경안+before/after 스크린샷→내가 폰에서 승인→배포.
  신뢰리스크 0. (배포=내가 직접 push, 롤백=git 1커밋=1수정 → revert)
- **C. 카톡 입력**: 고객은 이미 카톡으로 보냄. 초기엔 기존 1:1 카톡 + 내가 수동 중계
  (카카오 비즈채널 봇은 사업자인증·심사 필요해 후행).
- **B. 셀프서비스+가드레일**(나중): 고객 직접 포털 + "수정 가능 구역"만 AI가 건드리게 제약.
- 진화: A+C로 시작 → 고객 늘면 B의 가드레일로 점진 자동화.

### 전제 (합의됨)
1. 진짜 제품은 "AI 수정"이 아니라 "구독형 무제한 유지보수". AI=마진엔진.
2. (교정) 지인은 *시장*이 아니라 *세상에서 제일 싼 검증 채널*. 검증 기준 = 친구 vs 모르는 사람이
   아니라 **돈 내는가 + 사라지면 곤란한가**. → 지인에게 가격표 들이밀어 검증.
3. 제품의 생사 = "AI가 라이브 사이트를 안 망가뜨리는가". 사람게이트/수정구역제약 없는 auto-deploy
   금지(비전문가는 diff 못 읽고 롤백 못 함).

## 3. 검증 결과 (이번 주 과제 실행함)
- **demand gate 통과**: 미용실 원장님 + 진엽 형 회사 둘 다 월 구독 확답. 내가 가격 부르기 전에
  **"계좌 달라, 5만원씩 보내겠다"가 먼저 나옴** (= WTP 하한 5만원, 고객이 먼저 지갑 꺼냄).
- 실사용 빈도 = 고객당 **월 1회** 정도. ("무제한" 워딩 대비 저사용.)
- 결제수단 = 사업자 계좌이체 등으로 가능.
- 심리적 이슈: 친분이라 돈 받기 부담스러움 (→ 아래 가격전략에서 해소).

## 4. 가격 전략 → 상세: artifacts/02
시장 벤치마크(2026): 한국 STANDARD 월 10만, SEO포함 20만 / 미국 small biz $75~600+, 무제한은 상위티어.
유지보수 시장 **#1 불만 = "어디까지 포함이냐 모호함"**.
- **5만원은 시세의 절반 이하 = 저평가.** 단 지금(검증·소상공인·친구)엔 OK.
- **친구 2명 = 5만원 "파운딩 고객가" 평생 고정. 신규는 7~9.9만(시세보다 살짝 아래)부터 천장 실험.**
  → 가격 차별로 "친구 돈 받기 미안함"도 해소(친구=후하게, 모르는 고객=시장가).
- **모델 = 정액(flat). 건당 과금 절대 금지** (건당이면 "또 돈 나가나?" 마찰 부활 = status quo 회귀).
- **저사용+AI = 마진 99%** (헬스장/보험 구조). "무제한"이 심리적 가치(눈치 안 보고 맡김) = 핵심.
- **"무제한"의 경계 명문화**: 무제한=콘텐츠 수정 / 신규페이지·리디자인·기능=별도견적 → 시장 #1 불만
  을 차별점으로 뒤집고 업셀 통로 확보.
- 직설: **돈 받기 싫은 그 마음이 함정. 지금 자동이체로 받아라.** 공짜로 하면 검증도 못 하고
  결국 지치고 고객도 미안해서 요청 못 함.

## 5. 서비스 카탈로그 + 랜딩페이지(webmanager.co.kr) → 상세: artifacts/03
- **카탈로그 전략**: fork9(28년 종합에이전시) 베끼지 말 것. 너 무기 = "범위가 칼같이 투명".
  **포함(무제한 콘텐츠수정) / 운영(정기) / 별도견적(업셀) / 안함(정직)** 4분할.
- **랜딩 10섹션**: 히어로 → 문제공감(Before) → 솔루션 3단계(After) → 카탈로그 테이블 → 가격 →
  3자 비교(셀프/에이전시/3&D) → 신뢰요소(미용실·형회사 before/after+후기) → FAQ → CTA반복 → 푸터.
- **핵심 통찰**: **랜딩 카피 = 제품 스펙.** "카톡→당일 반영" 약속이 곧 MVP 백로그. 마케팅과 구현은
  한 몸. A+C MVP면 AI 자동화 전이어도 (수동 당일처리로) 그 약속 정직하게 이행 가능.

### 즉시 확인할 버그
- `www.webmanager.co.kr` = ECONNREFUSED (연결 거부), apex `webmanager.co.kr/` = 살아있음(403 to bots).
  → **www 서브도메인 라우팅 점검 필요** (홍보를 www로 하면 고객이 죽은 페이지 만날 수 있음).
